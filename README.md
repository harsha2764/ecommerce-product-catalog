# 🛒 E-Commerce Product Catalog

A modern, responsive, and production-ready **E-Commerce Product Catalog Web Application** built using **React + Vite**.  
This project demonstrates real-world frontend architecture, state management, routing, and deployment practices used in professional web development.

---


## ✨ Features

- 🧩 Modular React component architecture
- 🔀 Client-side routing using React Router
- 🛍️ Product listing and detailed product pages
- 🛒 Fully functional shopping cart system
- 💾 Persistent cart state (LocalStorage)
- 🎨 Modern UI with dark mode + glassmorphism design
- ⚡ Optimized performance with Vite build system
- 📱 Fully responsive design (mobile, tablet, desktop)
- 🚀 Production-ready deployment

---

## 🛠️ Tech Stack

- React.js
- Vite
- React Router DOM
- JavaScript (ES6+)
- CSS3 (Flexbox + Grid)
- LocalStorage API
